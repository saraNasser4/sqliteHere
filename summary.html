SELECT

It is for get something from something
  • SELECT trackid, name, composer, unitprice FROM tracks; You need to get x y z from db called tracks
  • SELECT * FROM tracks; You need to get all of the data from db called tracks

ORDER BY

since if you used the normal SELECT it won't be in order we use this
  • SELECT trackid, name, albumid FROM tracks ORDER BY albumid ASC; You need to get x y z from db called tracks the ordering from the small to bigger
    • You can remove ASC since it's the default or you can add DESC
  • SELECT name, milliseconds, albumid FROM tracks ORDER BY albumid, milliseconds DESC; Here it will order using ASC in the albumid first and then will check for the milliseconds
  • SELECT name, milliseconds, albumid FROM tracks ORDER BY 3, 2 DESC; You can sort the data using the column num instead of its name

DISTINCT

it allows you to remove the duplicate rows in the result
  • SELECT DISTINCT city FROM customers ORDER BY city; It will keep only one of each duplicate city
  • SELECT name, milliseconds, albumid FROM tracks ORDER BY albumid, milliseconds DESC; Here it will order using ASC in the albumid first and then will check for the milliseconds
  • SELECT name, milliseconds, albumid FROM tracks ORDER BY 3, 2 DESC; You can sort the data using the column num instead of its name
